### Setup Steps MAC

- Install rosetta - 
/usr/sbin/softwareupdate --install-rosetta --agree-to-license

- Change the chip - 
arch -x86_64 zsh

- Install Meteor - 
curl https://install.meteor.com/ | sh  

- meteor npm install npx browserslist@latest --update-db
- meteor npm i
- meteor npm i --save sharp
- meteor npm start
